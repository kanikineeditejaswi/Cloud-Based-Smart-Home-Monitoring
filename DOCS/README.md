🏠 Cloud-Based Smart Home Monitoring & Automation Platform
📌 Project Overview

The Cloud-Based Smart Home Monitoring & Automation Platform is a serverless AWS-based application that enables users to remotely monitor and control smart home devices through a responsive web dashboard. The platform leverages multiple AWS managed services to provide secure, scalable, and real-time smart home automation.

🚀 Features
💡 Control smart devices (Light, Fan, Door, Air Conditioner)
🌡️ Real-time Temperature Monitoring
💧 Live Humidity Monitoring
💨 Gas Detection Status
🚶 Motion Detection
📈 Live Temperature Analytics Chart
📝 Recent Activity Log
☁️ Serverless AWS Architecture
🔒 Secure IAM-based Access Control
⚡ REST API Integration using API Gateway
📊 CloudWatch Monitoring & Logging
